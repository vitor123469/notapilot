# Contributing

## Development Commands

- `pnpm dev` para desenvolvimento local.
- `pnpm lint` para validar padrao de codigo.
- `pnpm build` para validar compilacao/geracao de artefatos.

## Branch Convention

Use os prefixos abaixo para nomear branches:

- `feat/*`
- `fix/*`
- `chore/*`

## Pull Request Checklist

- [ ] Branch criada a partir da `main` atualizada.
- [ ] Escopo da mudanca esta claro no titulo e descricao.
- [ ] `pnpm lint` executado com sucesso.
- [ ] `pnpm build` executado com sucesso.
- [ ] Revisao humana solicitada antes do merge.
